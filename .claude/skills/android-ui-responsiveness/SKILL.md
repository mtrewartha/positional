# Android UI Responsiveness
This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they
relate to how it should handle android ui responsiveness.

## When to use this skill

- When creating Compose UI layouts that need to adapt to different screen sizes
- When implementing phone-first layouts that progressively enhance for larger screens
- When using standard breakpoints to adjust layouts for phone, tablet, and desktop
- When building fluid layouts with percentage-based widths and flexible containers
- When using sp/dp units instead of fixed pixels for better scalability
- When ensuring tap targets meet minimum size requirements (44x44px) for mobile
- When optimizing images and assets to reduce APK size for mobile performance
- When testing UI across multiple screen sizes and ensuring readability without zoom
