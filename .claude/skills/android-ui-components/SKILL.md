# Android UI Components
This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they
relate to how it should handle android ui components.

## When to use this skill

- When creating @Composable functions for UI components
- When designing reusable components with configurable parameters and sensible defaults
- When building complex UIs by composing smaller, simpler components
- When defining clear component interfaces and keeping implementation details private
- When managing component state and deciding when to lift state up
- When naming components with clear, descriptive names that indicate their purpose
- When keeping parameter counts manageable through composition or splitting components
- When documenting component usage, parameters, and providing examples
