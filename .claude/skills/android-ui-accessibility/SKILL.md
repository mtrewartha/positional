# Android UI Accessibility
This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they
relate to how it should handle android ui accessibility.

## When to use this skill

- When creating Composable functions that need appropriate semantics for assistive technologies
- When adding images or icons that require content descriptions
- When implementing keyboard navigation and visible focus indicators for interactive elements
- When choosing colors and ensuring sufficient contrast ratios (4.5:1 for normal text)
- When designing UI that conveys information beyond just color
- When adding labels to form inputs and interactive components
- When testing UI components with screen readers like TalkBack
- When working on any Compose UI code in .kt files that users will interact with
