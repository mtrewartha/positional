# Product Roadmap

## Current State

Positional has achieved its core mission with three primary features: location display (multiple
coordinate formats), digital compass (with magnetic declination), and solar times calculator. The
app is stable, privacy-focused, and successfully deployed on Google Play Store with both AOSP and
GMS flavors.

## Future Development Phases

These have not been determined yet.

> Notes
> - Features ordered by user value and alignment with minimalist privacy-focused mission
> - All features maintain strict privacy standards: no data storage, no tracking, no sharing without
    explicit user action
