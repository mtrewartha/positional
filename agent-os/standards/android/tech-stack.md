## Tech stack

### Framework & Runtime

- **Application Framework:** Android
- **Language/Runtime:** Kotlin
- **Build System & Dependency Manager:** Gradle

### Database & Storage

- **Relational Databases:** SQLite with Room as ORM
- **Key-value Stores:** Proto-based DataStore

### Testing & Quality

- **Test Frameworks:**
    - JUnit 4 (instrumented tests)
    - JUnit 5 (non-instrumented unit tests)
    - Kotest (assertions, matchers, and test style)
- **Linting/Formatting:** Detekt

### Deployment & Infrastructure

- **Automation:** Gradle
- **CI/CD:** GitHub Actions
- **Hosting:**
    - Google Play (production builds)

### Third-Party Services

- **Crash Reporting:** Firebase
