## Tech stack
### Framework & Runtime

- **Application Framework:** Android
- **Language/Runtime:** Kotlin in Android Runtime (ART)
- **Package Manager:** Gradle

### Frontend

- **UI Components:** Compose UI, Material Design components

### Database & Storage

- **Database:** SQLite
- **ORM/Query Builder:** AndroidX Room
- **User Preferences:** AndroidX DataStore

### Testing & Quality

- **Test Framework:** Kotest (specifically its BehaviorSpec style)
- **Linting/Formatting:** lint

### Deployment & Infrastructure

- **Hosting:** Google Play
- **CI/CD:** GitHub Actions

### Third-Party Services

- **Monitoring:** Firebase
