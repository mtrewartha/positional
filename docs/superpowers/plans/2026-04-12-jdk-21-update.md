# JDK 17→21 Update Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Update all JDK 17 references to JDK 21 so Metro 0.13.2's Gradle plugin resolves correctly, while cleaning up one duplicated constant and two stale comments.

**Architecture:** The project has a central `KotlinConfigurationConstants.kt` that defines the JVM version constants consumed by all convention plugins. Two files (`build-logic/convention/build.gradle.kts` and `app/build.gradle.kts`) cannot use those constants and must be updated separately. CI is updated independently.

**Tech Stack:** Kotlin, Gradle convention plugins, GitHub Actions

---

### Task 1: Update shared JVM version constants

**Files:**
- Modify: `build-logic/convention/src/main/kotlin/io/trewartha/positional/KotlinConfigurationConstants.kt`

This is the single source of truth consumed by `library.android.gradle.kts` and `library.jvm.gradle.kts` via `JVM_SOURCE_COMPATIBILITY` and `JVM_TARGET`. Changing it here propagates everywhere those constants are used.

- [ ] **Step 1: Update the constants**

Change both values in `KotlinConfigurationConstants.kt`:

```kotlin
val JVM_SOURCE_COMPATIBILITY = JavaVersion.VERSION_21

val JVM_TARGET = JvmTarget.JVM_21
```

- [ ] **Step 2: Commit**

```bash
git add build-logic/convention/src/main/kotlin/io/trewartha/positional/KotlinConfigurationConstants.kt
git commit -m "build: bump shared JVM version constants from 17 to 21 (POS-40)"
```

---

### Task 2: Update hardcoded values in build-logic convention build file

**Files:**
- Modify: `build-logic/convention/build.gradle.kts`

This file builds the convention plugin project itself, so it can't reference `KotlinConfigurationConstants.kt` — it must use hardcoded values.

- [ ] **Step 1: Update the hardcoded values**

In `build-logic/convention/build.gradle.kts`, change:

```kotlin
java {
    sourceCompatibility = JavaVersion.VERSION_21
    targetCompatibility = JavaVersion.VERSION_21
}

tasks.withType<KotlinJvmCompile>().configureEach {
    compilerOptions {
        jvmTarget = JvmTarget.JVM_21
        freeCompilerArgs.add("-Xcontext-parameters")
    }
}
```

- [ ] **Step 2: Commit**

```bash
git add build-logic/convention/build.gradle.kts
git commit -m "build: bump build-logic convention plugin JVM target from 17 to 21 (POS-40)"
```

### Task 3: Clean up library.android convention plugin and update app build file

**Files:**
- Modify: `build-logic/convention/src/main/kotlin/io/trewartha/positional/library.android.gradle.kts`
- Modify: `app/build.gradle.kts`

Two things happen here:
1. `library.android.gradle.kts` has hardcoded `JavaVersion.VERSION_17` on lines 18-19 that should use the shared `JVM_SOURCE_COMPATIBILITY` constant (the same file already uses it on lines 98-99). Replace those and remove the now-stale desugaring version comment.
2. `app/build.gradle.kts` has hardcoded `VERSION_17` / `JVM_17` plus the same stale comment.

- [ ] **Step 1: Fix library.android.gradle.kts**

Replace the `compileOptions` block (currently lines 14-21) in `library.android.gradle.kts`:

```kotlin
android {
    compileOptions {
        // https://developer.android.com/studio/write/java11-minimal-support-table
        sourceCompatibility = JVM_SOURCE_COMPATIBILITY
        targetCompatibility = JVM_SOURCE_COMPATIBILITY
        isCoreLibraryDesugaringEnabled = true
    }
```

- [ ] **Step 2: Fix app/build.gradle.kts**

Replace the `compileOptions` block (currently lines 55-61) in `app/build.gradle.kts`:

```kotlin
    compileOptions {
        // https://developer.android.com/studio/write/java11-minimal-support-table
        sourceCompatibility = JavaVersion.VERSION_21
        targetCompatibility = JavaVersion.VERSION_21
        isCoreLibraryDesugaringEnabled = true
    }
```

Replace the `kotlin` block (currently lines 124-137) in `app/build.gradle.kts`:

```kotlin
kotlin {
    compilerOptions {
        freeCompilerArgs.addAll(
            "-Xjvm-default=all",
            "-Xinline-classes",
        )
        jvmTarget = JvmTarget.JVM_21
        optIn.addAll(
            "androidx.compose.material3.windowsizeclass.ExperimentalMaterial3WindowSizeClassApi",
            "com.google.accompanist.permissions.ExperimentalPermissionsApi",
            "kotlin.time.ExperimentalTime",
        )
    }
}
```

- [ ] **Step 3: Commit**

```bash
git add build-logic/convention/src/main/kotlin/io/trewartha/positional/library.android.gradle.kts app/build.gradle.kts
git commit -m "build: replace hardcoded JVM 17 with 21 in library.android and app plugins (POS-40)"
```
